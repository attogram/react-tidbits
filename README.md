# react-tidbits

_Alpha Release_

A React component to display ever-changing text messages.

# Usage

```jsx harmony
const tidbits = [
  "array of tidbits",
  "another tidbit",  
];

<Tidbits
    interval="2000" // milliseconds
    tidbits={tidbits}
/>
```
